### Logo Best Practices

‚úÖ **Recommended Format:** PNG with transparent background  
‚úÖ **Recommended Size:** 200px wide x 60px tall (or similar ratio)  
‚úÖ **Max File Size:** 2MB  
‚úÖ **Also Accepts:** JPG, SVG formats

- **Transparent Background:** PNG format allows your logo to blend seamlessly with the header

- **Professional Design:** Consider hiring a designer for a polished look

- **Toggle Option:** Show or hide "Rifenburgh Landscaping" text next to logo
- **Located:** In admin panel under logo upload
- **Recommended:** Hide company name if your logo includes text

### How to Upload Hero Background

1. **Open Admin Panel**
2. **Scroll to "Hero Section Customization"**
3. **Click "Upload Hero Background"**
4. **Select a landscape photo** (your best work!)
5. **Image appears immediately** on homepage

### Hero Image Best Practices

‚úÖ **Recommended Format:** JPG (for photos)  
‚úÖ **Recommended Size:** 1920px x 1080px or larger  
‚úÖ **Max File Size:** 5MB  
‚úÖ **Orientation:** Landscape (horizontal)  
‚úÖ **Subject:** Showcase your best landscaping work  

### Tips for Great Hero Images

**Choose Images That:**
- Showcase your best completed projects
- Have good lighting and composition
- Are visually appealing and professional
- Represent the quality of your work
- Have a clear focal point

**Avoid Images With:**
- Too much clutter or busy backgrounds
- Poor lighting or low quality
- Text that might conflict with hero text
- Distracting elements
- Overly dark images (green overlay adds darkness)

**Pro Tips:**
- The hero has a green overlay (for brand consistency)
- Make sure your image subject is still visible under the overlay
- Bright, vibrant images work best
- Seasonal updates keep content fresh (spring flowers, fall colors, etc.)

## üìù Hero Text Customization

### How to Edit Hero Text

1. **Open Admin Panel**
2. **Scroll to "Hero Section Customization"**
3. **Find "Hero Text Content" subsection**
4. **Edit Main Headline** field
5. **Edit Subtitle/Description** field
6. **Click "Save Hero Text"** button
7. **See changes immediately** on homepage

### Default Text (for reference)

**Default Headline:**
```
Professional Landscaping Services in Germantown, NY
```

**Default Subtitle:**
```
Expert lawn care, mowing, striping, and land clearing services. 
Serving Germantown, Hudson, Red Hook & surrounding areas. 
Family-owned, reliable, and trusted by local homeowners.
```

### Hero Text Best Practices

**Main Headline:**
- Keep it concise (under 100 characters)
- Focus on your main service or value proposition
- Include your location (great for SEO)
- Make it benefit-focused

**Good Examples:**
- "Transform Your Outdoor Space - Germantown's Premier Landscaping"
- "Professional Lawn Care Services in Hudson Valley"
- "Your Trusted Landscaping Partner Since 2010"

**Subtitle/Description:**
- Expand on your headline (2-3 short sentences)
- Mention key services
- Include service area
- Add trust signals (family-owned, licensed, years of experience)
- Keep it under 200 characters for mobile readability

**Reset to Default:**
- If you want to go back to original text
- Click "Reset to Default" button
- Original text is restored instantly

## üíæ Storage & Persistence

### Where Are Customizations Saved?

All branding customizations are saved in **browser localStorage**:

- ‚úÖ **Logo** ‚Üí `customLogo`
- ‚úÖ **Logo Text Toggle** ‚Üí `showLogoText`
- ‚úÖ **Hero Background** ‚Üí `heroBackground`
- ‚úÖ **Hero Title** ‚Üí `heroTitle`
- ‚úÖ **Hero Subtitle** ‚Üí `heroSubtitle`

### Important Storage Notes

1. **Browser-Specific:** Changes only visible in the browser you used
2. **Not Shared:** Different browsers won't see your changes
3. **Persistent:** Data stays even after closing browser
4. **Backup Recommended:** Keep original files saved elsewhere
5. **Private Mode:** Don't use incognito/private browsing

### Storage Space

- **Total Available:** ~5-10MB depending on browser
- **Logo:** ~50-200KB (compressed)
- **Hero Image:** ~200-500KB (compressed)
- **Text:** Negligible space
- **Galleries:** Remaining space (~4-9MB)

## üêõ Troubleshooting

### Logo Issues

**Logo Not Appearing?**
- Check file size under 2MB
- Try PNG format
- Hard refresh (Ctrl+F5)
- Check browser console for errors

**Logo Looks Pixelated?**
- Upload higher resolution (at least 200px wide)
- Use PNG or SVG for sharpness
- Avoid stretching small images

**Logo Too Large/Small?**
- Max height is 50px
- Max width is 200px
- Logo auto-scales to fit
- Use horizontal orientation

### Hero Image Issues

**Image Not Showing?**
- Check file size under 5MB
- Use JPG or PNG format
- Clear browser cache
- Try different browser

**Image Looks Dark?**
- This is normal (green overlay for branding)
- Use brighter source images
- Increase image brightness before uploading

**Image Position Wrong?**
- Image centers by default
- Use landscape-oriented photos
- Main subject should be in center

### Hero Text Issues

**Text Not Saving?**
- Click "Save Hero Text" button
- Check both fields are filled
- Don't use incognito mode
- Check console for errors

**Text Too Long?**
- Keep headline under 100 characters
- Keep subtitle under 200 characters
- Test on mobile devices
